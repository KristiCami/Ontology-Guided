# 6.4 Αναφορά πλήρους ανάλυσης εκτελέσεων

Το αρχείο `runs/e1_to_e6_runs_analysis.md` αποτελεί την επίσημη και εκτενή τεχνική αναφορά για τα πειράματα E1–E6. Περιλαμβάνει:

- Αναλυτικούς πίνακες αποτελεσμάτων (F1, precision/recall, CQ pass rate).
- Μεθοδολογία ανάγνωσης των JSON αρχείων (`run_report.json`, `metrics_semantic.json`, `cq_results.json`, `repair_log.json`).
- Σχολιασμό ανά πείραμα, αιτίες αποκλίσεων και παρατηρήσεις για τα stop criteria.
- Συμπεράσματα και προτάσεις επόμενων βημάτων.

Για την τεκμηρίωση του Κεφαλαίου 6, το συγκεκριμένο αρχείο πρέπει να θεωρηθεί η πρωτεύουσα πηγή λεπτομερειών, ενώ τα υπόλοιπα έγγραφα του `docs/ch6/` λειτουργούν ως συνοπτική παρουσίαση και συζήτηση.
